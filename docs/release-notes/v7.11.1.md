# Whispering v7.11.1 - Recording Markdown Export

Whispering now writes recording markdown files only to the export folder you choose in settings. The appdata `recordings/` folder is used for audio blobs, while recording rows continue to live in the workspace database.

Legacy `.md` files in the appdata `recordings/` folder are left in place. You can delete them manually if you no longer need them. Whispering does not delete them automatically.
